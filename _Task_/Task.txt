
Зробить 1 сторінку на Webform або МVC з title, description, keywords.
	1. Дизайн не критично, можна самий найрпростіший.

	2. Поля вводу :
		Ім'я
		Прізвище
		Вік (роки)
		Професія.
			Де Ім'я, Прізвище - обов'язкові непусті поля, вік - числове поле від 0 до 100.
		
		Під ними кнопка "Відправити (javascript)" і "Відправити (Asp.Net)", 
		а нижче div, в який вставити результати відправленого у форматі таблиці 
			Ім'я : Прізвище : Вік :Професія :
		
		Після кнопки "Відправити" новий запис має добавлятись до списку

	3. Під ними textarea, в якому можна вводити html розмітку.
	   Нижче кнопка "Показати (javascript)" i "Показати (Asp.Net)", яка буде виводити цю розмітку в div в HTML

	4. Окремо написати SQL для таблиці Options (структура int id, nvarchar(50) name,int value)
	   Яка дозволить вибрати значення id які діляться на 3 без залишку

_________________

Функціонал (2,3) організувати двома cпособами - через javascript і через ASP.Net

До сторінки застосувати стилі: 1) через тег <style> або 2) через атрибут style.

До сторінки застосувати javascript: 1) через js файл або 2) через тег <script>

Не використовувати js, css фреймворки.

Сторінка має працювати однаково в Mozilla FF (остання версія 42), Chrome (остання версія 47), IE (10 та 11)
